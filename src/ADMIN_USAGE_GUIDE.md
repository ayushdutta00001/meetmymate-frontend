## âœ¨ Creating an Admin Account

### How to Create a New Admin Account:

#### Step 1: Access the Signup Page
1. On the Admin Login screen, you'll see the text: **"Don't have an account?"**
2. Click on **"Create Admin Account"** link below the login form

#### Step 2: Fill Out the Registration Form
You'll need to provide the following information:

**Required Fields:**
- **Full Name**: Your complete name (e.g., "John Doe")
- **Email Address**: Valid email (e.g., "john.admin@meetmymate.com")
- **Phone Number**: Contact number (e.g., "+91 98765 43210")
- **Admin Role**: Select from dropdown:
  - Admin
  - Super Admin
  - Content Moderator
  - Support Manager
- **Password**: Minimum 8 characters
- **Confirm Password**: Must match your password
- **Terms & Conditions**: Check the box to agree

#### Step 3: Form Validation
The form includes automatic validation:
- âœ… Email format check
- âœ… Phone number validation
- âœ… Password strength (minimum 8 characters)
- âœ… Password matching
- âœ… Required fields check

**Common Validation Errors:**
- "Full name is required"
- "Invalid email format"
- "Invalid phone number"
- "Password must be at least 8 characters"
- "Passwords do not match"
- "You must agree to the terms and conditions"

#### Step 4: Submit & Success
1. Click **"Create Admin Account"** button
2. You'll see a success screen with:
   - âœ… Green checkmark icon
   - "Account Created Successfully!" message
   - Your login credentials displayed
   - Automatic redirect to login (2 seconds)

#### Step 5: Login with New Account
- Use your registered email and password
